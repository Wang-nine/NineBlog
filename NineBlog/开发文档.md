# 2.项目结构

```
NineBlog/
├── .gradle/                          # Gradle 构建缓存目录
├── .idea/                            # IntelliJ IDEA 配置目录
├── .log/                             # 日志文件目录
├── gradle/                           # Gradle wrapper 文件目录
├── src/
│   ├── main/
│   │   ├── java/
│   │   │   └── com/
│   │   │       └── wang/
│   │   │           ├── aspect/       # AOP 切面类
│   │   │           ├── common/       # 公共类/工具类
│   │   │           ├── config/       # 配置类
│   │   │           ├── constant/     # 常量定义
│   │   │           ├── controller/   # 控制器层
│   │   │           ├── dto/          # 数据传输对象
│   │   │           ├── repository/   # 数据访问层
│   │   │           ├── service/      # 业务逻辑层
│   │   │           └── util/         # 工具类
│   │   │               └── NineBlogApplication.java  # Spring Boot 启动类
│   │   └── resources/
│   │       ├── static/               # 静态资源目录
│   │       ├── templates/            # 模板文件目录
│   │       └── application.properties # 应用配置文件
│   └── test/                         # 测试代码目录
├── .gitattributes                    # Git 属性配置
├── .gitignore                        # Git 忽略文件配置
├── build.gradle                      # Gradle 构建脚本
├── gradlew                           # Gradle wrapper 脚本 (Unix)
├── gradlew.bat                       # Gradle wrapper 脚本 (Windows)
├── HELP.md                           # 帮助文档
├── settings.gradle                   # Gradle 设置文件
└── 开发文档.md                        # 项目开发文档
```
